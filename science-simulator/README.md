# Science Simulator: Diversity of Materials

A static web application simulating physical properties of materials (Strength, Flexibility, Buoyancy, Waterproofness, Transparency) for Primary 3 Science students.

## Features
- **Interactive Simulations**: Test materials for various properties.
- **Visual Feedback**: Animations and visual cues for test results.
- **Static Build**: Can be run without a backend server.

## Running Locally
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`

## Static Deployment
This project has been configured to run as a static HTML site.
1. Build the project: `npm run build`
2. The static files will be in the `dist/` directory.
3. You can open `dist/index.html` directly in a browser (ensure `vite.config.js` has `base: './'`).

## Controls
- **Fullscreen**: Toggle fullscreen mode using the button in the top right.
- **Back**: Return to the main repository using the Back button.
